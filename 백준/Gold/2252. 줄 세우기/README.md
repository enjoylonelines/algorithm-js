# 2252. 줄 세우기

**출처:** 백준 Online Judge  
**난이도:** Gold III  
**분류:** 위상 정렬  
**링크:** https://www.acmicpc.net/problem/2252

---

## 문제 설명

N명의 학생들을 키 순서대로 줄을 세우려고 한다.

각 학생의 키를 직접 재지 않고, 두 학생의 키를 비교한 결과만 주어진다. 이 정보를 바탕으로 학생들을 줄 세운 결과를 출력하라.

비교 결과가 모든 학생들의 순서를 완전히 결정하지 않을 수도 있다. 이 경우 가능한 줄 세우기 중 아무거나 출력한다.

---

## 입력 형식

첫째 줄에 N(1 ≤ N ≤ 32,000)과 M(1 ≤ M ≤ 100,000)이 주어진다.

다음 M개의 줄에 걸쳐 A와 B가 주어진다. 이는 A번 학생이 B번 학생보다 앞에 서야 한다는 뜻이다.

---

## 출력 형식

줄을 세운 결과를 공백으로 구분하여 출력한다.

---

## 예시

### 예시 1

**입력:**
```
3 2
1 3
2 3
```

**출력:**
```
1 2 3
```

### 예시 2

**입력:**
```
4 4
1 2
1 3
2 4
3 4
```

**출력:**
```
1 2 3 4
```

### 예시 3

**입력:**
```
5 4
1 2
2 3
3 4
4 5
```

**출력:**
```
1 2 3 4 5
```

---

## 제약조건

- 1 ≤ N ≤ 32,000
- 1 ≤ M ≤ 100,000
- 학생 번호는 1 이상 N 이하의 자연수
- 모든 비교 결과는 사이클이 없음을 보장한다
- 시간 제한: 2초
- 메모리 제한: 128 MB
