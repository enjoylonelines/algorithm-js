# 12865. 평범한 배낭

**출처:** 백준 Online Judge  
**난이도:** Gold V  
**분류:** 다이나믹 프로그래밍, 배낭 문제  
**링크:** https://www.acmicpc.net/problem/12865

---

## 문제 설명

이 문제는 아주 평범한 배낭 문제이다.

한 달 후면 국가 행사가 열린다. 수강생들은 배낭에 물건을 담아 행사에 참여한다.  
배낭에는 최대 K 무게까지만 넣을 수 있다.

N개의 물건이 있으며, 각 물건은 무게 W와 가치 V를 가진다.  
배낭에 넣을 수 있는 물건들의 가치의 합의 **최댓값**을 구하라.  
물건은 쪼갤 수 없으며 각 물건은 한 번씩만 사용할 수 있다.

---

## 입력 형식

첫째 줄에 물건의 수 N과 배낭의 최대 무게 K가 주어진다.

다음 N개의 줄에 각 물건의 무게 W와 가치 V가 주어진다.

---

## 출력 형식

한 줄에 배낭에 넣을 수 있는 물건들의 가치합의 최댓값을 출력한다.

---

## 예시

### 예시 1

**입력:**
```

4 8
3 6
5 12
```

**출력:**
```
14

```

4 6

16
```

### 예시 3

```
5 15
3 4

5 6

```
21
```

## 제약조건

- 1 ≤ N ≤ 100
- 1 ≤ K ≤ 100,000
- 1 ≤ W ≤ 100,000
- 0 ≤ V ≤ 1,000
- 시간 제한: 2초
- 메모리 제한: 512 MB
