# 1158. 요세푸스 문제

**출처:** 백준 Online Judge  
**난이도:** Silver IV  
**분류:** 구현, 자료구조, 큐  
**링크:** https://www.acmicpc.net/problem/1158

---

## 문제 설명

요세푸스 문제는 다음과 같다.

1번부터 N번까지 N명의 사람이 원을 이루면서 앉아있고, 양의 정수 K가 주어진다. 순서대로 K번째 사람을 제거한다. 한 사람이 제거되면 남은 사람들로 이루어진 원을 따라 계속 진행한다. N명의 사람이 모두 제거될 때까지 이 과정을 반복한다.

제거되는 순서를 요세푸스 순열이라고 한다. N과 K가 주어지면 요세푸스 순열을 구하는 프로그램을 작성하시오.

---

## 입력 형식

첫째 줄에 N과 K가 빈칸을 사이에 두고 순서대로 주어진다.

---

## 출력 형식

예제와 같이 요세푸스 순열을 출력한다.

---

## 예시

### 예시 1

**입력:**
```
7 3
```

**출력:**
```
<3, 6, 2, 7, 5, 1, 4>
```

### 예시 2

**입력:**
```
5 2
```

**출력:**
```
<2, 4, 1, 5, 3>
```

### 예시 3

**입력:**
```
1 1
```

**출력:**
```
<1>
```

---

## 제약조건

- 1 ≤ K ≤ N ≤ 5,000
- 시간 제한: 2초
- 메모리 제한: 256 MB
