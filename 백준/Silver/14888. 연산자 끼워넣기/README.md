# 14888. 연산자 끼워넣기

**출처:** 백준 Online Judge  
**난이도:** Silver I  
**분류:** 구현, 완전탐색, 백트래킹  
**링크:** https://www.acmicpc.net/problem/14888

---

## 문제 설명

N개의 수로 이루어진 수열 A1, A2, ..., AN이 주어진다.  
수열의 두 인접한 수 사이에 연산자를 하나씩 끼워 넣어 수식을 만든다.  
연산자는 덧셈(+), 뺄셈(-), 곱셈(×), 나눗셈(÷) 네 종류이며, 각 연산자의 개수가 주어진다.  
연산자 우선순위는 무시하고 수식을 앞에서부터 차례대로 계산한다.

나눗셈은 몫만 취하며, **피제수가 음수일 경우 C++ 기준 정수 나눗셈**을 따른다 (음수를 양수로 나눈 뒤 음수 부호를 붙인다).

수식의 결과의 **최댓값**과 **최솟값**을 구하라.

---

## 입력 형식

첫째 줄에 수의 개수 N이 주어진다.

둘째 줄에 A1, A2, ..., AN이 주어진다.

셋째 줄에 덧셈(+)의 개수, 뺄셈(-)의 개수, 곱셈(×)의 개수, 나눗셈(÷)의 개수가 주어진다.

---

## 출력 형식

첫째 줄에 수식 결과의 최댓값을, 둘째 줄에 최솟값을 출력한다.

---

## 예시

### 예시 1

**입력:**
```
2
5 6
0 0 1 0
```

**출력:**
```
30
30
```

### 예시 2

**입력:**
```
3
3 4 5
1 0 1 0
```

**출력:**
```
35
17
```

### 예시 3

**입력:**
```
6
1 2 3 4 5 6
2 1 1 1
```

**출력:**
```
54
-24
```

---

## 제약조건

- 2 ≤ N ≤ 11
- 1 ≤ Ai ≤ 100
- 연산자 개수의 합은 N-1
- 항상 최댓값과 최솟값은 -10억 이상 10억 이하
- 시간 제한: 2초
- 메모리 제한: 512 MB
