# 1940. 주몽

**출처:** 백준 Online Judge  
**난이도:** Silver IV  
**분류:** 투 포인터, 정렬  
**링크:** https://www.acmicpc.net/problem/1940

---

## 문제 설명

주몽은 철기군을 양성하기 위한 갑옷을 만들려고 한다. 갑옷을 만드는 데 적합한 재료들은 given 번호가 있으며, 갑옷은 두 가지 재료로 만드는데 두 재료의 번호를 합쳐서 M이 되어야 한다.

N개의 재료에서 두 재료를 골라 합이 M이 되는 경우의 수를 구하여라. 단, 한 번 사용한 재료는 다시 사용할 수 없으며, 같은 번호의 재료가 여러 개 있을 수 있다.

---

## 입력 형식

첫째 줄에 재료의 개수 N이 주어진다.  
둘째 줄에 갑옷을 만드는 데 필요한 수 M이 주어진다.  
셋째 줄에 N개의 재료들이 나타내는 각각의 번호들이 공백으로 구분되어 주어진다.

---

## 출력 형식

첫째 줄에 가능한 경우의 수를 출력한다.

---

## 예시

### 예시 1

**입력:**
```
6
9
2 7 4 1 5 3
```

**출력:**
```
2
```

### 예시 2

**입력:**
```
4
10
1 9 2 8
```

**출력:**
```
2
```

### 예시 3

**입력:**
```
5
7
3 4 6 2 5
```

**출력:**
```
2
```

---

## 제약조건

- 1 ≤ N ≤ 15,000
- 1 ≤ M ≤ 10,000,000
- 각 재료의 번호는 1 이상 100,000 이하의 자연수이다.
- 시간 제한: 2초
- 메모리 제한: 128 MB
