# 1744. 수 묶기

**출처:** 백준 Online Judge  
**난이도:** Silver III  
**분류:** 그리디 알고리즘, 정렬  
**링크:** https://www.acmicpc.net/problem/1744

---

## 문제 설명

길이가 N인 수열이 주어졌을 때, 그 수열의 합을 구할 수 있는 방법이 있다.

- 수열의 어떤 두 수를 **묶어** 더하는 대신 곱한 값으로 계산할 수 있다.
- 묶인 수는 서로 곱해져 수열의 합에 더해진다.
- 각 수는 최대 한 번만 묶을 수 있다.
- 묶이지 않은 수는 그냥 수열의 합에 더한다.

이때 수열의 합을 최대로 만드는 값을 구하라.

---

## 입력 형식

첫째 줄에 수열의 크기 N이 주어진다.

둘째 줄부터 N개의 줄에 수열의 각 수가 하나씩 주어진다.

---

## 출력 형식

수열의 합의 최댓값을 출력한다.

---

## 예시

### 예시 1

**입력:**
```
4
3
4
-2
0
```

**출력:**
```
12
```

### 예시 2

**입력:**
```
2
1
1
```

**출력:**
```
2
```

### 예시 3

**입력:**
```
4
-1
-5
10
8
```

**출력:**
```
85
```

---

## 제약조건

- 1 ≤ N ≤ 50
- -1,000 ≤ 수열의 각 수 ≤ 1,000
- 수열의 각 수는 정수
- 시간 제한: 2초
- 메모리 제한: 128 MB
