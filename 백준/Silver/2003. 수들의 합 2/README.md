# 2003. 수들의 합 2

**출처:** 백준 Online Judge  
**난이도:** Silver IV  
**분류:** 두 포인터  
**링크:** https://www.acmicpc.net/problem/2003

---

## 문제 설명

N개의 수로 된 수열 a[1], a[2], ..., a[N] 이 있다.

이 수열의 i번째 수부터 j번째 수까지의 합 a[i] + a[i+1] + ... + a[j]가 M이 되는 경우의 수를 구하여라.

---

## 입력 형식

첫째 줄에 N(10 ≤ N ≤ 10,000)과 M(1 ≤ M ≤ 300,000,000)이 주어진다.

다음 줄에 N개의 수가 주어지며, 각 수는 1 이상 30,000 이하의 자연수이다.

---

## 출력 형식

첫째 줄에 경우의 수를 출력한다.

---

## 예시

### 예시 1

**입력:**
```
4 2
1 1 1 1
```

**출력:**
```
3
```

### 예시 2

**입력:**
```
10 5
1 2 3 4 2 5 3 1 1 2
```

**출력:**
```
3
```

### 예시 3

**입력:**
```
5 15
1 2 3 4 5
```

**출력:**
```
1
```

---

## 제약조건

- 10 ≤ N ≤ 10,000
- 1 ≤ M ≤ 300,000,000
- 수열의 각 원소는 1 이상 30,000 이하의 자연수
- 시간 제한: 0.5초
- 메모리 제한: 32 MB
