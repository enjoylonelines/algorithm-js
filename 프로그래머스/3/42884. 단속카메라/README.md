# [level 3] 단속카메라 - 42884

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/42884)

### 문제 설명

고속도로를 이동하는 모든 차량이 고속도로를 이용하면서 단속용 카메라를 한 번은 만나도록 카메라를 설치하려고 합니다.

고속도로를 이동하는 차량의 경로 routes가 매개변수로 주어질 때, 모든 차량이 한 번은 단속용 카메라를 만나도록 하려면 최소 몇 대의 카메라를 설치해야 하는지를 return 하도록 solution 함수를 완성하세요.

### 입출력 형식

- 입력: 2차원 배열 `routes` — `routes[i]`는 i번째 차량의 [진입 지점, 진출 지점]
- 출력: 필요한 최소 카메라 대수 (정수)

### 제한사항

- 차량의 대수는 1대 이상 10,000대 이하입니다.
- routes에는 차량의 이동 경로가 포함되어 있으며 routes[i][0]에는 차량이 고속도로에 진입한 지점, routes[i][1]에는 차량이 고속도로에서 나간 지점이 적혀 있습니다.
- 차량의 진입/진출 지점에 카메라가 설치되어 있어도 카메라를 만난 것으로 간주합니다.
- 차량의 진입 지점, 진출 지점은 -30,000 이상 30,000 이하입니다.

### 입출력 예

| routes | result |
|---|---|
| [[-20, -15], [-14, -5], [-18, -13], [-5, -3]] | 2 |
| [[0, 10]] | 1 |
| [[-2, -1], [1, 2], [3, 4]] | 3 |

### 입출력 예 설명

**예 1**: -5 지점에 카메라를 설치하면 두 번째, 네 번째 차량이 카메라를 만납니다. -15 지점에 카메라를 설치하면 첫 번째, 세 번째 차량이 카메라를 만납니다. 총 2대가 필요합니다.

**예 2**: 차량이 1대이므로 경로 위 아무 지점에 1대만 설치하면 됩니다.

**예 3**: 세 차량의 경로가 서로 겹치지 않으므로 각각 1대씩, 총 3대가 필요합니다.
