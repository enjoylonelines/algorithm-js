# [level 3] 순위 - 49191

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/49191)

### 문제 설명

n명의 권투선수가 권투 대회에 참여했고 각각 1번부터 n번까지 번호를 받았습니다. 권투 경기는 1대1 방식으로 진행이 되고, 만약 A 선수가 B 선수보다 실력이 좋다면 A 선수는 B 선수를 항상 이깁니다. 심판은 주어진 경기 결과를 가지고 선수들의 순위를 매기려 합니다. 하지만 몇몇 경기 결과를 분실하여 정확하게 순위를 매길 수 없습니다.

선수의 수 n, 경기 결과를 담은 2차원 배열 results가 매개변수로 주어질 때 정확하게 순위를 매길 수 있는 선수의 수를 return 하도록 solution 함수를 완성해주세요.

### 입출력 형식

- 입력: 정수 `n`, 2차원 배열 `results` — `results[i]`는 [이긴 선수 번호, 진 선수 번호]
- 출력: 순위를 정확하게 매길 수 있는 선수의 수 (정수)

### 제한사항

- 선수의 수는 1명 이상 100명 이하입니다.
- 경기 결과는 1개 이상 4,500개 이하입니다.
- 모든 경기 결과에는 모순이 없습니다.

### 입출력 예

| n | results | result |
|---|---|---|
| 5 | [[4, 3], [4, 2], [3, 2], [1, 2], [2, 5]] | 2 |
| 3 | [[1, 2], [2, 3]] | 3 |
| 4 | [[1, 2]] | 0 |

### 입출력 예 설명

**예 1**: 2번 선수는 1, 3, 4번 선수에게 패배했고 5번 선수에게 승리했으므로 4위가 확정입니다. 5번 선수는 2번 선수에게 패배했으므로 나머지 선수들에게도 모두 패배한 것이 되어 5위가 확정입니다. 나머지 선수들의 순위는 확정할 수 없으므로 답은 2입니다.

**예 2**: 1번이 2번을 이기고 2번이 3번을 이겼으므로 1위: 1번, 2위: 2번, 3위: 3번으로 모든 선수의 순위가 확정됩니다.

**예 3**: 1번과 2번의 상대 전적만으로는 3번, 4번 선수와의 관계를 알 수 없어 어떤 선수의 순위도 확정할 수 없습니다.
